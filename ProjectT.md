# 🚀 PROJECT T : Next-Gen Financial Data Visualization Engine

> **"The Future is not written, It is Rendered."** > 본 프로젝트는 단순한 금융 웹사이트가 아닙니다. 거대 자본이 흉내 낼 수 없는 **제로 디펜던시(Zero-Dependency)** 기반의 초고속 실시간 금융 데이터 시각화 엔진입니다.

---

## 🌟 프로젝트 개요 (Overview)

기존 금융 시스템의 한계를 뛰어넘기 위해 밑바닥(Low-level)부터 독자 설계된 **Deep-Tech 시각화 엔진**입니다. 수천 개의 금융 객체를 지연 없이(Zero-Latency) 실시간으로 렌더링하며, 복잡한 숫자를 인간의 본능이 즉각 인지할 수 있는 시청각 정보로 변환합니다.

---

## 🛠 핵심 기술 경쟁력 (Core Technology)

### 1. 제로 디펜던시 & 로우 레벨 최적화

* **No Framework:** React, Vue, Chart.js 등 외부 라이브러리를 단 한 줄도 사용하지 않았습니다.
* **Pure Performance:** WebGL/Canvas 및 메모리 관리를 C++ 수준으로 직접 제어하여, 브라우저 환경에서도 워크스테이션급 퍼포먼스를 구현합니다.
* **In-Memory Architecture:** 상용 DB(Redis 등)를 쓰지 않고 10년 전부터 독자 설계한 인메모리 아키텍처를 사용하여 데이터 처리 속도를 극대화했습니다.

### 2. 패킷 다이렉트 캡처 (Direct Data Feed)

* 일반적인 API 호출 방식이 아닌, 데이터 패킷 자체를 직접 낚아채는 기술을 사용합니다.
* 중계업자를 거치지 않는 **'데이터 채굴'** 영역의 기술로, 타 서비스가 따라올 수 없는 데이터 신선도를 보장합니다.

---

## 💎 주요 서비스 기능 (Key Features)

### 📊 머니 플로우 (Money Flow) : "돈의 길목을 지키다"

* **실시간 전수 조사:** 시장 전체 종목의 매수/매도 체결 금액을 0.1초 단위로 합산합니다.
* **스마트 머니 추적:** 지금 이 순간, 거대 자본이 어떤 섹터와 종목으로 유입되는지 실시간 순위를 매겨 '답안지' 같은 데이터를 제공합니다.

### 🌪 아트모스피어 (Atmosphere) : "시장 분위기를 본능으로 읽다"

* **클러스터링 시각화:** 수만 개의 종목 데이터를 상관관계에 따라 점으로 표시합니다.
* **직관적 인지:** 숫자를 해석하기 전, 데이터가 뭉치고 흩어지는 모양을 통해 시장의 '공포'와 '탐욕'을 본능적으로 감지합니다.

### 🎙 AI 오감 매매 (Sensory Trading)

* **시청각 동기화:** 화면을 보지 않아도 AI가 시장의 급변 상황을 음성으로 브리핑합니다.
* **전투기 조종석 경험:** 중요 신호 발생 시 즉각적인 사운드와 시각 효과를 통해 트레이더가 뇌의 계산 없이 즉각 반응할 수 있도록 돕습니다.
